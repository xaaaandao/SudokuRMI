Sistemas distrbuídos - UTFPR - CM - (2017-2)


### Execução

```
$ git clone https://github.com/xaaaaaandao/SudokuRMI.git

Abra três terminais, em um deles digite o comando abaixo:
$ make registry

No outro terminal, digite:
$ make server

E no último terminal, digite:
$ make client
```
